Welcome to เห้ย! ลืมเปลี่ยนภาษา

เว็บไซต์นี้ ได้แรงบันดาลใจมาจากเว็บ Lullar ที่แปลงข้อความพิมพ์ผิดจากอังกฤษเป็นไทยได้อย่างเดียวเวลาลืมกดเปลี่ยนภาษา เว็บนี้เลยทำให้สามารถเปลี่ยนประโยคหรือข้อความที่พิมพ์ผิดจากภาษาอังกฤษให้เป็นภาษาไทย รวมทั้งเปลี่ยนประโยคหรือข้อความที่พิมพ์ผิดจากภาษาไทยให้เป็นภาษาอังกฤษ เวลาลืมกดเปลี่ยนภาษาได้อัตโนมัติ

ต้องบอกก่อนเลยว่า project นี้ได้พลังของเจ้า ChatGPT มาช่วย และ modify เองเพิ่มด้วย คิดเล่น ๆ ว่าถ้ามีเวลาว่าง ในอนาคต อยากลองใช้ NLP มาช่วย detect ความหมายด้วยว่า ข้อความไหนที่มีความหมายอยู่แล้ว ไม่ต้อง convert เป็นอีกภาษา ขอเวลาไปศึกษาก่อน เหมือนจะมี data set Thai Dictionary และ English Dictionary รวมทั้ง Open Access NLP ให้ใช้ แต่เป็น Python ไม่แน่ใจจะ import มาใช้กับ JavaScript ยังไง ถ้าเป็นไปได้ อยากให้รองรับ Layout Keyboard ทุกภาษาในโลกเลย 5555 (ฝันไกลมาก) ใครมี idea แนะนำมาได้เลยค่าาาา
